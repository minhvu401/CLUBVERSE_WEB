👤 Quản lý người dùng Admin Only 3 API ▶
GET
Lấy danh sách tất cả người dùng
GET /users
Chỉ ADMIN mới có quyền — xem toàn bộ user trong hệ thống
DEL
Vô hiệu hóa tài khoản người dùng
DELETE /users/{id}/deactivate
Soft deactivate — tài khoản bị khóa nhưng dữ liệu giữ nguyên
PATCH
Kích hoạt lại tài khoản
PATCH /users/{id}/reactivate
Mở khóa tài khoản đã bị vô hiệu hóa trước đó
📋 Quản lý đơn đăng ký vào CLB Club/Admin 5 API ▶
GET
Xem danh sách đơn đăng ký của 1 club
GET /applications/club/{clubId}?status=PENDING|APPROVED|REJECTED...
Filter theo trạng thái đơn
GET
Xem chi tiết đơn đăng ký
GET /applications/{id}
PATCH
Duyệt đơn + gửi lịch phỏng vấn
PATCH /applications/{id}/approve
Body: interviewDate (required), interviewLocation, interviewNote
PATCH
Từ chối đơn đăng ký
PATCH /applications/{id}/reject
Body: rejectionReason
PATCH
Quyết định cuối sau phỏng vấn
PATCH /applications/{id}/final-decision
Body: decision (accepted | declined). Nếu accepted → tự động thêm vào thành viên CLB
🏛️ Quản lý thành viên CLB Club Account 6 API ▶
GET
Xem danh sách thành viên CLB
GET /club-members/club/{clubId}
Filter: status, role, search, sortBy
GET
Xem chi tiết 1 thành viên
GET /club-members/club/{clubId}/member/{memberId}
DEL
Xóa thành viên khỏi CLB
DELETE /club-members/club/{clubId}/remove
Body: userId, reason (bắt buộc)
PUT
Cập nhật vai trò thành viên
PUT /club-members/club/{clubId}/update-role
Body: userId, newRole (admin | moderator | member)
GET
Thống kê thành viên CLB
GET /club-members/club/{clubId}/statistics
GET
Export danh sách thành viên
GET /club-members/club/{clubId}/export
Filter theo status: active | inactive | all
🎯 Quản lý sự kiện & bài viết Club Account 8 API ▶
POST
Tạo sự kiện mới
POST /events
title, description, location, time, maxParticipants (required)
PATCH
Cập nhật sự kiện
PATCH /events/{id}
DEL
Xóa mềm sự kiện
DELETE /events/{id}/soft
DEL
Xóa vĩnh viễn sự kiện
DELETE /events/{id}/hard
PATCH
Khôi phục sự kiện đã xóa
PATCH /events/{id}/restore
GET
Danh sách người tham gia sự kiện
GET /events/{id}/participants
POST
Check-in người tham gia
POST /events/{id}/check-in/{userId}
DEL
Hủy check-in
DELETE /events/{id}/check-in/{userId}/undo
